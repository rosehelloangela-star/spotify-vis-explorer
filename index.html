Spotify 是如何得到这些数值的？

1
音频信号分析 — 上传的 MP3/WAV 文件被分解为频谱图，提取响度、频率分布、节拍等原始信号特征。
2
Echo Nest 机器学习模型 — Spotify 于 2014 年收购的 Echo Nest 团队训练了多组分类/回归模型，把原始信号特征映射成 0–1 之间的语义分数。
3
人工标注校准 — 模型最初用人类打分数据训练（如"这首歌你会在派对上放吗？"），之后随着更多用户行为数据不断迭代校准。
4
结果写入 Audio Features API — 最终数值通过 Spotify Web API 公开，这也是 Kaggle 数据集里这些列的来源。
